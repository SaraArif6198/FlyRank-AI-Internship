## Before you post — do these yourself (nobody else can)

1. **Add the voice card to your Claude Project.** Open `Portfolio Build — Sara Arif` on claude.ai → Set custom instructions → paste the block from `voice_card.md`. Screenshot it.
2. **Generate the PDF.** Open `index.html` → drop the Claude Project screenshot into §5 Evidence → Print / Save as PDF. (Text is double-click editable if you want to tweak anything first.)
3. **Check the paraphrased testimonial in Case 1.** It's built from your memory of what the site manager said (time saved, no more disputes, didn't disappear). If his actual words were different, fix them — it's labeled as a paraphrase either way.
4. **The read-aloud edit.** Read every case out loud. Cut anything you wouldn't say to the DG Cement site manager across a table. This is the assignment's actual test.
5. **Post** `thread_post.md` content + the PDF to your track thread.

## Pass/revise criteria mapping

- *A framed case for each piece the sitemap calls for* → deliverable covers Proof + all six Work builds + bio + CTA (sitemap FL-03 v2)
- *Three beats, could only be your project* → problem / decisions / outcome in each; specifics (26-day formula, 17-pattern guardrail) are unique to you
- *Sounds like a specific person, before/after shows it* → voice card enforced; before/after included
- *One audience, one action, tight* → every case faces the ops manager; single CTA: email a scoped inquiry
