## Before you post

1. **(Recommended) Re-run the ladder live in your Claude Project** — paste each rung's prompt in order and compare what YOU get against the excerpts here; swap in your own outputs anywhere they differ (all text in `index.html` is double-click editable). The ladder's structure, layers, and notes stay the same — the evidence gets even more yours.
2. **Read the four notes per rung out loud** — they're graded on describing *output* changes; if any note reads like a prompt description to you, sharpen it.
3. **Print / Save as PDF** from `index.html`.
4. **Post** `thread_post.md` content + the PDF to your track thread.

## Pass/revise criteria mapping

- *Six runs, one named layer each* → Rung 0–5, layer named in every heading; the ladder strip on page 2 shows the whole climb
- *Notes describe output changes* → "what improved in the output" cell in every rung's grid
- *One honest "made it worse" moment* → Rung 4 (example layer) — photocopied phrases, transplanted facts; removed at Rung 5
- *Final prompt works for a stranger* → §2, fully `{brace}`-parameterized, with the warning about the failed layer
